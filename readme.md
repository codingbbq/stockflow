
# StockFlow

StockFlow is a web app for managing inventory.
- Public users can browse stock items.
- Authenticated users can request inventory.
- Admins can manage stock, approve requests, and track changes.

---

## Frontend Technologies Used

- **React 18** (with TypeScript)
- **Vite** (for fast development)
- **Radix UI** UI Components
- **Tailwind CSS** (styling)
- **TanStack Query** (React Query for data fetching/caching)
- **Wouter** (client-side routing)
- **React Hook Form** + **Zod** (form handling and validation)
- **Lucide React** (icons)

---

## Backend Tech Stack

- **Express.js** (with TypeScript)
- **Drizzle ORM** (database queries)
- **PostgreSQL** (hosted on Neon)
- **Passport.js** (authentication with OpenID Connect via Replit)
- **Express Sessions** (session management, stored in PostgreSQL)
- **Zod** (API validation)
